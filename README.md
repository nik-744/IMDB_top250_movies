# Exploratory Data Analysis (EDA) of IMDb's Top 250 Movies
This repository contains an Exploratory Data Analysis (EDA) conducted on IMDb's Top 250 movies dataset. The dataset comprises ratings, reviews, genres, directors, and other relevant information about the top-rated movies on IMDb.

# Analysis Highlights
The EDA explores various aspects of the dataset, including:

Distribution of IMDb ratings
Correlation between IMDb rating and Metascore
Most frequent genres among the top-rated movies
Directors with the highest-rated movies
Runtime distribution
and more.

# Technologies Used
The analysis was performed using Python and popular data science libraries including Pandas, Matplotlib, and Seaborn. Jupyter Notebook was utilized for the interactive analysis.
